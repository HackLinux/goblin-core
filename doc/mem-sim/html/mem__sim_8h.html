Detailed Description

GoblinCore-64 Memory Interface Simulation Library Header FIle.

The MEM_SIM library is a standard C library that provides functional simulation support for experimenting with various memory interface modules in association with the GoblinCore-64 architecture.

Initializes the internal MEM_SIM state with the necessary hardware configuration options.

Parameters
*msimis a pointer to a valid memsim structure. Must not be null.
ifaceis the interface type: MEMSIM_BASIC or MEMSIM_HMC
algis the coalescing algorithm: MEMSIM_SIMPLE or MEMSIM_EXP
task_groupsis the number of task groups on the socket
task_procsis the number of task procs in a task group
tasksis the number of tasks per task proc
g_slotsis the number of interface slots at the group interface
s_slotsis the number of interface slots at the socket interface
a_slotsis the number of interface slots at the amo interface [socket]
ga_slotsis the number of interface slots at the global addressing interface
optis a OR'd set of memsim options: MEMSIM_AMO, MEMSIM_GA
Returns
0 on success, nonzero otherwise
